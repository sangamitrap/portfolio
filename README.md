# Sangamitra P - Portfolio Website

A modern, professional, and interactive portfolio website showcasing Sangamitra's skills, projects, achievements, and professional journey.

## Features

### 🎨 Design & UI
- **Professional Color Scheme**: Navy blue and blue gradient theme
- **Modern Typography**: Clean Inter font family
- **Responsive Design**: Fully responsive across all devices
- **Smooth Animations**: Parallax effects, fade-ins, and micro-interactions
- **Interactive Elements**: Hover effects, skill progress bars, and card animations

### 📱 Sections
1. **Hero Section**: Eye-catching introduction with contact information
2. **Education Timeline**: Visual timeline of academic journey
3. **Technical Skills**: Categorized skills with animated progress bars
4. **Projects Showcase**: Interactive project cards with tech stacks
5. **Achievements**: Visual achievement cards with image placeholders
6. **Certificates**: Professional certificate display
7. **Internship Experience**: Detailed internship information
8. **Coding Profiles**: Links to competitive programming profiles
9. **Contact Section**: Professional contact information

### ⚡ Interactive Features
- **Smooth Scrolling**: Seamless navigation between sections
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Scroll Animations**: Elements fade in as you scroll
- **Skill Progress Bars**: Animated skill level indicators
- **Project Card Tilt**: 3D hover effect on project cards
- **Active Navigation**: Dynamic highlighting of current section
- **Copy to Clipboard**: Click email to copy functionality
- **Typing Effect**: Animated name typing in hero section

## 🚀 Getting Started

1. **Clone or Download** the project files
2. **Open `index.html`** in your web browser
3. **Navigate** through the portfolio sections

## 📸 Adding Your Images

To add your personal images to the achievements section:

1. Replace the placeholder URLs in the achievement cards:
   ```html
   <img src="your-image-path.jpg" alt="Achievement Description">
   ```

2. Recommended image sizes:
   - Achievement images: 300x200 pixels
   - Profile image: 150x150 pixels (circular)

3. Supported formats: JPG, PNG, WebP

## 🔧 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1e3a8a;
    --secondary-color: #3b82f6;
    --accent-color: #60a5fa;
    /* ... other variables */
}
```

### Fonts
Change the font family in `styles.css`:
```css
body {
    font-family: 'Your-Font', sans-serif;
}
```

### Content
Update the content directly in `index.html`:
- Personal information
- Project descriptions
- Skills and proficiency levels
- Achievement details
- Contact information
